# onDestroy and watchers

  • Directive onDestroy
  • Unbinding watchers

Une directive qui est détruite lève un event destroy.

Donc dans le controleur de la directive il faut écouter cet event :

scope.$on('$destroy', function() {
  console.log("destroy");
});
