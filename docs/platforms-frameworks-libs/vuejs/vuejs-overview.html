# vue-cli

vuejs/vue-cli (opens new window)

Current available templates include:

  • webpack (opens new window) - A full-featured Webpack + vue-loader setup with hot reload, linting, testing & css extraction.
  • webpack-simple (opens new window) - A simple Webpack + vue-loader setup for quick prototyping.
  • browserify (opens new window) - A full-featured Browserify + vueify setup with hot-reload, linting & unit testing.
  • browserify-simple (opens new window) - A simple Browserify + vueify setup for quick prototyping.
  • pwa (opens new window) - PWA template for vue-cli based on the webpack template.
  • simple (opens new window) - The simplest possible Vue setup in a single HTML file.

# single file components

Single File Components : vuejs.org (opens new window)

A single file containing html, js and css :

<template>
    <p>{{ greeting }} World !</p>
</template>

<script>
module.exports = {
    data: function() {
        return {
            greeting: 'Hello'
        }
    }
}
</script>

<style scoped>
p {
    font-size: 2em;
    text-align: center;
}
</style>

# http management

Retiring vue-resource (opens new window)

As Vue users, many of you may have used vue-resource (opens new window) for handling ajax requests in your Vue applications.

However, over time we have come to the conclusion that an "official ajax library" is not really necessary for Vue.

  • axios/axios (opens new window)
