# XML

Extensible Markup Language

Exemple :

```xml

<phrase titre-livre="XML idiot">
    <paragraphe>
        <chapitre>
            <toto>        
                <livre>
                    <titi>Informatique sans bon sens n’est que ruine de l'âme et 
perte de temps.</titi>
                </livre>
            </toto>
        </chapitre>
    </paragraphe>
</phrase>

```
